# Financial KPI Analysis Dashboard

## Author
**Arshpreet Singh**

## Project Overview
This project focuses on analyzing financial performance using Power BI. The dashboard provides key insights into sales, profit, discounts, and product performance through interactive visualizations and KPI tracking.

## Objective
The objective of this project is to monitor business performance using Key Performance Indicators (KPIs) and support strategic decision-making through data visualization.

## Dataset
Financial Sample Dataset

## Key Performance Indicators (KPIs)
- Gross Sales
- Total Sales
- Total Profit
- Total Discounts
- Total Units Sold
- Profit Margin (%)

## Dashboard Visualizations
- Monthly Sales Trend
- Sales by Product
- Sales by Segment
- Sales by Country
- Profit by Discount Band

## Interactive Filters
- Country
- Year
- Segment
- Product

## Key Insights
- Government segment generated the highest sales contribution.
- Paseo emerged as the top-performing product.
- Sales peaked during the final quarter of the year.
- Profit margins remained healthy across most discount categories.
- Country-wise analysis highlighted strong revenue performance in key markets.

## Tools Used
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

## Business Impact
This dashboard enables users to:
- Track financial performance efficiently.
- Identify profitable products and markets.
- Monitor sales trends over time.
- Support strategic business decisions using data-driven insights.

## Conclusion
The Financial KPI Analysis Dashboard provides a comprehensive view of business performance through KPIs, charts, and interactive filters. It helps stakeholders make informed decisions by transforming raw financial data into meaningful insights.
